Description

This project is a MySQL database designed to track student service requests (such as IT help tickets). It stores students, staff members, request categories, and ticket status updates. It supports creating requests, assigning them, and viewing open or closed tickets.

The focus was designing tables with clear relationships and writing SQL queries that return useful results.

Skills Learned

  • Designing tables with primary keys and foreign keys
  • Writing SQL queries (SELECT, INSERT, UPDATE, DELETE)
  • Using JOINs to combine related tables
  • Maintaining data integrity with relationships
