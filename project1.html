Description

This project is a command-line food ordering system written in C. It lets a user view a menu, select items, choose quantities, and see an updated total. At the end, the program prints a receipt that lists each item, quantity, and the final cost.

The goal was to practice basic programming logic, user input handling, and organizing code in a clear way.

Skills Learned

  • Using variables, conditionals, and loops in C
  • Working with arrays and simple menu systems
  • Validating user input
  • Breaking code into functions
